# HR Analytics Executive Dashboard

A comprehensive data analytics project focused on monitoring employee attrition, workforce utilization, and talent acquisition pipeline dynamics using **SQL** and **Power BI**.

---

## 📌 Project Overview & Key Tracks
This dashboard acts as a strategic decision-making tool for corporate HR leaders, deep-diving into four core operational areas:
1. **Employee Attrition Analysis:** Uncovering the root causes of employee turnover and tracking rolling attrition velocity.
2. **Workforce Utilization Analysis:** Monitoring capacity, tracking billable vs. non-billable hours, and assessing burnout indicators.
3. **Hiring Trend Analysis:** Evaluating the efficiency of the talent acquisition pipeline (Time-to-Hire, Cost-per-Hire).
4. **Department Performance Reporting:** Mapping employee productivity metrics and performance ratings against business outcomes.

---

## 🛠️ Tech Stack & Architecture
- **Data Querying & Engineering:** SQL Server (Data Cleaning, Transformation, and Schema Structuring)
- **Analytical Metrics Engine:** DAX (Data Analysis Expressions)
- **Visualization Suite:** Power BI Desktop UI Canvas
- **Data Source Prototyping:** MS Excel

---

## 📂 Project Structure
- 📁 **Dataset/**: Contains raw and structured dummy data configurations.
- 📁 **SQL_Scripts/**: Production-ready SQL queries for database views and pipelines.
- 📁 **PowerBI_Report/**: Interactive `.pbix` dashboard application files.
- 📁 **Screenshots/**: Visual captures of finalized tracking panels.
