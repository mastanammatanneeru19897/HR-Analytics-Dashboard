# HR Analytics Dashboard

### 📊 Project Overview
This Power BI dashboard analyzes HR data to track employee performance, department salaries, and headcount details.

### 🛠️ Tech Stack & Tools
* **Data Source:** GitHub Raw CSV Dataset
* **Transformation:** Power Query Editor
* **Data Modeling:** Star Schema (`Dim_Date` table linked via Active/Inactive relationships)
* **Visualization:** Power BI Desktop
